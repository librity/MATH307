Polynomial InterpolationÂ¶

import numpy as np
import scipy.linalg as la
import matplotlib.pyplot as plt

Given \(d+1\) data points \((t_0,y_0), \dots , (t_d,y_d)\) there exists a unique polynomial of degree \(d\)

\[ p(t) = c_0 + c_1 t + \cdots + c_d t^d \]

such that \(p(t_k) = y_k\) for each \(k=0,\dots,d\). The coefficients \(c_k\) are unknown and each data point gives us an equation

\[\begin{split} \begin{align*} c_0 + c_1t_1 + \cdots + c_d t_1^d &= y_1 \\ c_0 + c_1t_2 + \cdots + c_d t_2^d &= y_2 \\ & \vdots \\ c_0 + c_1t_d + \cdots + c_d t_d^d &= y_d \end{align*} \end{split}\]

This yields a system of equations \(A \boldsymbol{c} = \boldsymbol{y}\)

\[\begin{split} \begin{bmatrix} 1 & t_1 & \cdots & t_1^d \\ 1 & t_2 & \cdots & t_2^d \\ \vdots & \vdots & \ddots & \vdots \\ 1 & t_d & \cdots & t_d^d \end{bmatrix} \begin{bmatrix} c_0 \\ c_1 \\ \vdots \\ c_d \end{bmatrix} = \begin{bmatrix} y_0 \\ y_1 \\ \vdots \\ y_d \end{bmatrix} \end{split}\]

The matrix \(A\) is called a Vandermonde matrix.

Example 1Â¶

Let the data points \((-1,0),(\frac{1}{2},2),(2,-1)\) be given. We want to find a polynomial which interpolates these points. Letâ€™s create the system using numpy.vander and solve with scipy.linalg.solve.

t = np.array([-1,0.5,2])
y = np.array([0,2,-1])
A = np.vander(t,increasing=True)

print(A)

[[ 1.   -1.    1.  ]
 [ 1.    0.5   0.25]
 [ 1.    2.    4.  ]]

c = la.solve(A,y)

print(c)

[ 1.88888889  0.77777778 -1.11111111]

The solution corresponds to the coefficients \(c_0=\frac{17}{9}\), \(c_1=\frac{7}{9}\) and \(c_2=-\frac{10}{9}\).

Lets plot this:

T = np.linspace(-2,3,100)
Y = c[0] + c[1]*T + c[2]*T**2
plt.plot(T,Y,'b-',t,y,'r.',markersize=10)
plt.grid(True)
plt.show()

Example 3Â¶

What happens to the condition number as \(N\) increases when \(t_0,t_1,\dots,t_N\) are \(N+1\) equally spaced points from 0 to 1? That is, \(t_k = k/N\), \(k=0,\dots,N\).

C = []
Nmax = 11
for N in range(2,Nmax):
    t = np.linspace(0,1,N+1)
    A = np.vander(t)
    c = np.linalg.cond(A)
    C.append(c)
plt.figure(figsize=(8,5))
plt.plot(range(2,Nmax),C,'b.-',markersize=10)
plt.title('Condition Number of Vandermonde matrix of size N+1')
plt.grid(True)
plt.xlabel('N')
plt.show()

C[-1]

115575244.59700233

The condition number for \(N=10\) is around \(10^8\). Yikes!

Example 4Â¶

Letâ€™s interpolate the points \(\sin(\pi t_k)\) for \(t_k = k/N\) for \(N=15\).

N = 15
t = np.linspace(0,1,N+1)
y = np.sin(np.pi*t)
A = np.vander(t,increasing=True)
c = la.solve(A,y)
T = np.linspace(0,1,100)
Y = sum([c[k]*T**k for k in range(0,len(c))])
plt.plot(T,Y,'b-',t,y,'r.',markersize=10)
plt.grid(True)
plt.show()

The interpolating polynomial is very nice! But what happens if we introduce some random noise in the data.

N = 15
t = np.linspace(0,1,N+1)
noise = 0.005*np.random.randn(t.size)
y = np.sin(np.pi*t) + noise
A = np.vander(t,increasing=True)
c = la.solve(A,y)
T = np.linspace(0,1,200)
Y = sum([c[k]*T**k for k in range(0,len(c))])
plt.plot(T,Y,'b-',t,y,'r.',markersize=10)
plt.grid(True)
plt.show()

Yikes! The interpolating polynomial is very sensitive to small changes in the \(y\) values! Thatâ€™s because the condition number is large!

What if we use a different interpolation methodÂ¶

Lets take Lagrange interpolation.

from scipy import interpolate

N = 15
t = np.linspace(0,1,N+1)
y = np.sin(np.pi*t)
T = np.linspace(0,1,100)
f = interpolate.lagrange(t, y)
Y = f(T)
plt.plot(T,Y,'b-',t,y,'r.',markersize=10)
plt.grid(True)
plt.show()

N = 15
t = np.linspace(0,1,N+1)
noise = 0.005*np.random.randn(t.size)
y = np.sin(np.pi*t) + noise
T = np.linspace(0,1,200)
f = interpolate.lagrange(t, y)
Y = f(T)
plt.plot(T,Y,'b-',t,y,'r.',markersize=10)
plt.grid(True)
plt.show()

Still bad! So we need to do something different! What about interpolating with polynomial of certain degree between each two points. However then we need to think about how to make the overall polynomial nice and continues.
